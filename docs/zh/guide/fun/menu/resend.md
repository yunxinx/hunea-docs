# /resend

`/resend` 用来把你之前发送过的消息重新填回输入框。

当你想复用以前发过的某条提示词、改一改再发，或者接着之前的草稿继续写，就可以用它。输入 `/resend` 确认后，会打开一个全屏的消息历史面板，标题是 `Message history`，你从中挑选想要回填的那条消息即可。

也可以在没有其他面板挡住时，直接按 `Ctrl + R` 打开同一份面板。和斜杠命令略有不同：`Ctrl + R` 打开时**不会清掉**输入框里已有的草稿；而通过菜单输入 `/resend` 时，输入框会被命令本身占掉，确认后通常是空的。

## 列表概览

面板列出的是**全局**的用户消息历史，不是“只看当前会话里的用户消息”。也就是说：你在别的项目目录里发过的内容，只要还被保留在这份历史里，这里也能找回来。

需要注意：

- 只记录**你发出去的用户输入**，不会出现助手回复
- 纯空白内容不会入库
- 连续两条正文完全相同的发送，相邻重复那条通常不会再记一遍

每一行大致会展示：

1. 相对时间（比如 `now`、`5m`、`2h`、`1d`）
2. 该条用户消息的摘要文本（列表里是截断展示；真正回填/复制时会用完整内容）

标题行会显示当前位置，例如 `Message history (2 of 10)`，方便判断“现在在第几条 / 一共多少条”。

加载时可能会先显示 `Loading message history...`；如果还没有任何可回填的历史，则会显示 `No sent messages yet`。

> 列表按时间从旧到新排列，默认光标会落在**最新**的一条。打开后通常可以直接 `Enter` 回填最近一次发送。

## 怎么操作

底部会有一行快捷键提示，常用操作如下：

- `↑` / `↓`，或者 `j` / `k`：上下移动选中项
- `←` / `→`，或者 `h` / `l`：按页翻动（历史多的时候比较有用）
- `/`：进入搜索模式，继续键入即可过滤列表
- `Enter`：把当前选中的消息**回填到输入框**
- `c`：复制当前选中消息的完整文本
- `Space`：打开当前选中消息的**完整内容预览**
- `Esc`：关闭面板；若正在搜索，则先退出搜索，再按一次才关闭

也可以用鼠标左键点击某一行，把光标移过去；点击本身不会触发回填。

搜索会匹配消息全文（大小写不敏感）。搜不到时会提示 `No messages match search`。搜索模式下：

- `Backspace`：删字
- `Ctrl + U`：清空当前搜索内容
- 普通字母会当作搜索词本身，例如 `c`、`j`/`k` 不会再触发复制或移动

在预览界面里：

- `Esc` 或 `Space`：退回列表
- `←` / `→` / `↑` / `↓`，以及 `h` / `j` / `k` / `l`：翻页查看更长内容
- `c`：仍然可以直接复制当前预览的那一条

预览里**没有** `Enter` 回填；要回填，需要先退回列表再按 `Enter`。

## 回填之后会发生什么

选中并按 `Enter` 后，Hunea 会：

1. 关闭面板
2. 把选中消息的**完整正文**写入输入框（光标一般会移到末尾）
3. **不会自动发送**；你可以继续编辑，再自己按发送

如果打开面板时输入框里还有未发送草稿（典型是 `Ctrl + R` 路径），确认回填前会先尽量把这份草稿记进消息历史，再换成你选中的那条。这样“写到一半却改去复用旧消息”时，半成品草稿不容易丢。

需要注意：这是**回填输入框**，不是重新发送、也不是把会话回溯到某条旧消息。会话树本身不会因为 `/resend` 而改动。

列表摘要可能因终端宽度被截断，但回填和 `c` 复制走的都是完整文本，不是屏幕上看到的那一行省略号内容。复制成功/失败时的提示，与其他复制路径一致（类似 `Selection copied` / `Copy selection failed`）。

## 历史会保留多少

这份全局历史有条数上限，配置项是：

```toml
[tui]
message_history_limit = 100
```

- 默认 `100`
- 可配范围大致是 `100` 到 `1000`
- 超限后会优先丢掉更旧的条目，保留较新的

数据落在 Hunea 的数据目录里（一般是 `~/.config/hunea/`），并且是**跨项目**共享的全局历史，而不是按当前工作区单独一份。

## 一点使用建议

- 只想快速复用“刚才那条”：打开后光标通常已在最新一条，直接 `Enter` 即可。
- 历史很长时，优先用 `/` 搜关键词，比纯靠上下翻更省事。
- 输入框里已经写了一半、又想对照旧消息：用 `Ctrl + R` 打开，草稿会留着；确认回填前也会尽量把草稿先记进历史。
- 只想带走文本、不想回填输入框：在列表或预览里按 `c` 复制即可。

如果你其实是想**从当前会话里挑用户/助手消息拷出去**，更适合用 [`/copy`](/guide/fun/menu/copy.html)，而不是 `/resend`。
