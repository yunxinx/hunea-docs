# fenced 代码块无背景回退样式统一

- **日期**：2026-07-17
- **提交**：`fix(tui): 统一 fenced 代码块无背景回退样式`
- **版本**：`0.11.1-alpha.6`

本次更新统一了对话区中纯文本代码块和高亮失败代码块的渲染样式：

1. **`text` / `txt` / `plain` / `plaintext` 按无语言代码块处理**  
   这些语言标识符不再套用带高亮的代码块外壳，渲染效果与裸 ` ``` ` 一致。
2. **高亮失败不再保留背景色**  
   当语法高亮失败时，现在与无语言代码块采用相同样式，不再残留多余的背景色。

本次变更仅影响对话区 Markdown 渲染的视觉效果，不涉及配置项修改。
