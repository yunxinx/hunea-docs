# Hunea Docs

[Hunea](https://github.com/yunxinx/hunea) 官方文档站点（[hunea.cc](https://hunea.cc)）。

Hunea 是基于 Rust + Ratatui 构建的终端 AI 助手客户端。

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

生产构建输出目录为 `doc_build`（可用于 Cloudflare Pages 等静态托管）。

## Preview

```bash
npm run preview
```
